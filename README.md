# 🎨 Frontend Engineer

> From Figma frame to production component in seconds

A Vienna content profile for frontend engineers. Includes quick actions for design-to-code conversion, accessibility audits, performance debugging, E2E test generation, and responsive design.

## What's included

- **Quick Actions** — Convert Figma to React, audit accessibility, debug re-renders, generate E2E tests, analyze bundle size, create responsive variants
- **Skills** — Getting started guide for Vienna profiles
- **Plugins** — Linear for issue tracking

## Install

Fork this profile in Vienna:

```
vienna://profile/github.com/tryvienna/profile-frontend-engineer
```

Or go to **Settings → Profiles → Fork** and paste:
```
https://github.com/tryvienna/profile-frontend-engineer.git
```

## Customize

After forking, this profile lives at `~/.vienna/profiles/frontend-engineer/`. Edit any file to make it yours — add quick actions, swap plugins, tweak skills.

## Share

Push your customized profile to your own GitHub repo and share the `vienna://profile/...` link. Others can fork your version in one click.
